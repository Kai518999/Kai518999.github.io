Machine Learning



what is machine learning

Machine learning is a subfield of Artificial Intelligence which is broadly defined as the capability of machine to imitate intelligent human behavior. Artificial Intelligence systems are used to perform complex tasks in a way that is similar to how humans solve problems which allows for automations. It was defined in the 1959 by AI pioneer Arthur Samuel as "the field of study that gives computers the ability to learn without explicity being programmed."

There are Four subcatagories of machine learning these are:

Supervised Machine Learning
These models are trained with a labeled data set which allows the models to learn and grow more accurate over time. For example an algorithm would be trained with pictures of dogs and other things, all labelled by humans and the machine would learn ways to identify pictures of dogs on its own.Supervised is the most common type used today.

Unsupervised Machine Learning
in this section the model looks for patterns in unlabeled data.For example this means if you gave it the data of people buying coats during the whole year. It will then look through the whole data and see that coats generally get bought more closer to the winter seasons of the year. This type of model is mostly used in going through online sales data and identify different types of clients making purchases

Reinforcement Machine Learning
The way this model works is through trial and error to take the best action by establishing a reward system. Reinforcement learning can train models to play games or train autonomous vehicles to drive by telling the machine when it made the right decisions,which helps it learn over time what actions it should take.

Semi-Supervised Machine Learning
This area takes a different approach than the other two areas by interacting with an environment built by people to learn different optimal behaviours. This model of machine learning works from rewards and penalties.
